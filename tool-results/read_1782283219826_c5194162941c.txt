     1→# 狼人杀游戏项目 - 工作日志

     3→## 项目概述
     4→构建一个移动端狼人杀手机游戏，支持用户与AI对战，包含语音输入、卡通风格UI、多种角色和套餐配置。
     5→
     6→## 技术栈
     7→- Next.js 16 + TypeScript + Tailwind CSS 4 + shadcn/ui
     8→- Zustand (游戏状态) + z-ai-web-dev-sdk (AI玩家决策/语音)
     9→- 单页面应用，视图状态切换 (menu/setup/game/result)

   204→Work Log:
   205→- 【问题分析】用户反馈每次都是狼人。
   206→  原因1: 新手局2狼/6人=33%概率当狼人，短期随机性偏差导致体感"总是狼人"。
   207→  原因2: 狼人夜晚行动印象深刻，平民/女巫等被动角色容易被遗忘(幸存者偏差)。
   208→  代码逻辑本身随机正确，但体验上用户无法控制角色。
   209→
   210→- 【新功能】角色偏好选择 (SetupScreen + engine + store):

   213→  - engine.initPlayers 增加 preferredRole 参数，支持4种模式:
